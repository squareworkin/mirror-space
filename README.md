# AI Mental Health Support Application
This project is an AI-based mental health support application that allows users to interact with an AI chatbot, maintain a personal journal, and track their mood and mental well-being.

## Features
- AI-based chatbot for emotional support
- Journaling feature for recording thoughts and feelings
- Mood tracking and pattern insights
- Calm and sleep support modules 

## Project Structure
client/ – frontend user interface  
server/ – backend APIs, services, and models  

## Purpose
The goal of this project is to provide users with a safe digital space where they can reflect on their emotions, receive supportive responses, and monitor their mental health patterns.

## Status
Partial implementation (~25–30%) for academic submission.
